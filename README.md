# How to use:

to proceed to installation, type in your terminal:

> git clone https://github.com/duduclx/graphina.git

> cd graphina

> chmod a+x basic-install.sh

> ./basic-install.sh

# to remove the installer, run:

> cd ~

> rm -R graphina
